# DESIGN.md

## 气质与意象

**赛博暗夜中的数字工坊**：一位全栈设计工程师的个人空间，如深夜实验室般沉稳而充满技术张力。视觉锚点——深黑画布上电光蓝线条勾勒出的数字流动感，仿佛数据在暗夜中穿行。整体气质是冷静、专业、未来感，拒绝温暖与柔和，用冷光与硬边传递技术实力。

## 配色方案

| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 主背景 | `#06080f` | 深空夜幕，几乎纯黑带微量蓝调 |
| 卡片/区块背景 | `#0d1117` | 略浅的炭黑，提供层次感 |
| 电光蓝（强调色） | `#00d4ff` | 电路板走线、霓虹灯管、数据流光 |
| 蓝色辅助 | `#0ea5e9` | 蓝色光晕外层、渐变过渡 |
| 主文字 | `#ffffff` | 标题、重要信息 |
| 次文字 | `#94a3b8` | 正文、说明文字（slate-400） |
| 弱文字 | `#475569` | 编号、辅助标注（slate-600） |
| 边框 | `rgba(0,212,255,0.15)` | 电光蓝低透明度，科技感边界 |
| 绿色功能色 | `#22c55e` | SECURE 等安全标识 |

## 字体排版

- **字体族**：Inter（西文）+ 系统默认中文字体（PingFang SC / Microsoft YaHei）
- **标题**：font-bold / font-extrabold，无衬线，方正硬朗
- **正文**：font-normal，轻量无衬线
- **排版节奏**：大标题 48-64px → 区块标题 24-32px → 正文 14-16px → 辅助 12px，层级分明
- **编号风格**：章节编号用 01/02/03... 浅灰色大号数字，标识视觉节奏

## 动效与交互

- **运动性格**：冷峻、精确，不拖泥带水
- **hover 卡片**：边框从暗变亮蓝（0.15→0.5 透明度），微微上移 2-4px + 阴影加深
- **按钮 hover**：电光蓝底色亮度提升 15%，添加 box-shadow 蓝色辉光
- **导航链接 hover**：白色→电光蓝，0.2s ease
- **滚动出现**：元素从下方 20px 淡入，0.6s ease-out
- **装饰光效**：蓝色发光线条用 CSS box-shadow / text-shadow 模拟，不用 canvas

## 页面结构

1. **导航栏**：固定顶部，半透明深色毛玻璃，左侧 Logo+姓名，中间导航链接，右侧功能图标
2. **Hero 首屏**：左文字右视觉，左侧核心标语+CTA按钮，右侧用 CSS+SVG 构建科技感视觉区（发光圆环+浮动卡片+星点背景）
3. **核心能力**：4列网格卡片，每卡含编号+图标+标题+技能列表
4. **项目展示**：2×2 网格卡片，每卡含项目截图占位+类型标签+简介
5. **工作流程**：6节点横向时间轴，蓝色连线串联，右侧理念引用框
6. **关于与联系**：3列布局——个人简介+技能标签、经历时间轴、联系方式+二维码占位
7. **页脚**：3列——版权、标语、社交图标

## 设计禁忌

- 禁止使用暖色系（橙、粉、金）作为主色或强调色
- 禁止圆角过大（max 12px），保持硬朗科技感
- 禁止使用手写体、衬线体作为 UI 字体
- 禁止大面积纯白区域，所有背景必须有深色基调
- 禁止过度动画（弹跳、旋转、闪烁），保持克制
- 禁止使用默认的蓝紫渐变（科技蓝+紫色），只用单一电光蓝
